# Roadmap

This roadmap is intentionally public and practical. It is not a promise of delivery dates.

## Next

- Add UI/e2e coverage for **Update from file**, multiple certificate import, and certificate remove/clear-all flows.
- Add a workspace health panel that explains what is stored in the workspace versus Electron `userData`.
- Improve mock scenario status labels for editor state, saved file state, and running runtime state.
- Improve REST, WebSocket, and gRPC response detail drawers for long payloads.
- Add richer `.proto` workflows, including import path management, reflection support, and proto validation.
- Add scenario-based testing for saved requests, examples, metadata, and mock responses.

## Later

- Add OpenAPI import/export workflows.
- Add team-oriented features such as workspace templates, shared examples, and review-friendly exports.
- Add more benchmark reporting formats and comparison reports.
- Add optional command palette support for power-user actions without expanding the logo menu.
