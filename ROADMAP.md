# Roadmap

This roadmap is intentionally public and practical. It is not a promise of delivery dates.

## Next
- Add the first automated test suite for Layang.
- Add unit tests for workspace storage, proto parsing, request execution, mocks, docs generation, and CLI commands.
- Add integration tests for opening workspaces, validating saved requests, running mock servers, and generating reports.
- Add richer `.proto` workflows, including import path management, reflection support, and proto validation.
- Add scenario-based testing for saved requests, examples, metadata, and mock responses.
- Add CI runner for validating workspaces and running saved scenarios.
- Add better mock server workflows, including dynamic responses, delays, errors, and streaming mocks.
- Stabilize the WebSocket beta introduced in `1.0.0-rc.2`, including workspace schema, mocks, benchmarks, and generated docs.

## Later

- Add OpenAPI/REST module.
- Add team-oriented features such as workspace templates, shared examples, and review-friendly exports.
- Add more benchmark reporting formats and comparison reports.
